Asteroid Bot,

I am an advanced intelligent bot for web-scrapping, finding/searching info and more!

AsteroidDiscuss

Privacy.

Asteroid is open-source, we just listen to user commands or messages starting with '/' in the telegram group or the personal chat.

Searching.

Asteroid can search for a query, keywords on the Internet which is mainly powered by Wikipedia and Google search.

Text to audio.

Asteroid can translate text input into an audio file in a moment.

Fast.

Asteroid responds in over 1 second typically to any message or commands you send to the bot.

Filters Hate-speech.

Bot just listens to commands, however, if you like to filter abusive/hate speech in you're Telegram group then you'll need to promote it to do so. The bot knows more than 1000+ abusive words.

Fun.

One of the reasons you should consider adding it because the bot is fun to use, especially in tech-related groups; searching for something useful on the Internet.

The Asteroid Project.

Asteroid is an open-source and non-profit project developed and maintained by ColorCubes; Fell free to use it.
